Implementace

*name jednoznačně identifikuje příslušnou složku v rámci zadání, například:

  • Requirements - servisní požadavky
  • Manager - správce města
  • Ticket - hlášení
  • Technicians/repairs - servisní technici
  1. app/Http/Controllers/{name} - hlavní řídicí kód
  2. app/Http/Middleware/{name}
  3. app/Services/{name} - některé doplňkové funkce
  4. resources/views/{name} - frontend
  5. app/Http/Requests/{name} - žádosti
  6. app/Models/{name} - modely a některé příslušné doplňkové funkce
  7. database/migrations/{name} - vytvoření tabulek v databázi
  8. routes/web.php - popis cest

Instalace

Stručný popis:

  • softwarové požadavky: PHP 8, Laravel 9, Composer, MariaDB
  • Založení databázi a instalace:
    1. Založit databázi pomocí MariaDB(nebo použít jinou databázi)
    2. Vytvořit Laravel projekt a přidat potřebné soubory
    3. Specifikovat databází v souboru .env
    4. Pouštět composer install
    5. Pouštět php artisan migrate
    6. Pouštět nmp install
    7. Pouštět nmp run build
    8. Pouštět nmp run dev
    9. Pouštět php artisan storage:link
    10. Pouštět php artisan serve
